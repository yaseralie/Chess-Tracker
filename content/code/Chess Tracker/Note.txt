
- Install library
pip install -r requirements.txt

- Install GTK+ for Windows Runtime Environment Installer
https://github.com/tschoonj/GTK-for-Windows-Runtime-Environment-Installer/releases
Download: gtk3-runtime-3.24.31-2022-01-04-ts-win64.exe
Jalankan dan install (biarkan path default, biasanya ke C:\Program Files\GTK3-Runtime Win64)
Setelah selesai, tambahkan path ini ke environment variable:
	C:\Program Files\GTK3-Runtime Win64\bin
Tutup dan buka ulang Command Prompt.
Lalu test:
	python -c "import cairocffi; print('Cairo OK!')"

- Run Python
python calibrate_manual_oriented.py --rotate 270
python cv_chess_play.py
